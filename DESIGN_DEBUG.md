## 디버깅 방법

### Chrome DevTools 체크
1. F12 → Elements 탭
2. Computed 스타일 확인
3. CSS 우선순위 검사

### 일반적인 문제 해결
1. `!important` 남용 → 특정성 조정
2. 캐시 문제 → Ctrl+F5 강새로고침
3. 모바일 뷰포트 → 메타태그 확인

### 성능 검사
1. Lighthouse 점수 확인
2. CLS (Cumulative Layout Shift) 최소화
3. 이미지 최적화 상태
